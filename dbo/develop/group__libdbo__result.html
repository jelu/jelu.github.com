typedef libdbo_result_t*(* libdbo_result_list_next_t)(void *data, int finish)

Function pointer for walking a libdbo_result_list.

Function pointer for walking a libdbo_result_list. The backend handle specific data is supplied in `data` and setting `finish` to non-zero tells the backend that we are finished with the libdbo_result_list.

Parameters:
[in]dataa void pointer for the backend specific data.
[in]finishan integer that if non-zero will tell the backend that we are finished with the result list.
Returns:
A pointer to the next libdbo_result_t or NULL on error.
